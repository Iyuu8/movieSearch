    What is yuuZ

    yuuZ is a lightweight movie & TV search app that lets you instantly find posters, ratings, and summaries with a clean, responsive design powered by the OMDb API.

    This project was built to practice real-world API integration and enhance interactive UI development with HTML, CSS, and JavaScript. yuuZ focuses on delivering a smooth user experience with animations and a minimalistic design for easy movie browsing.

    Why Was yuuZ Built

    As a passionate developer, I created yuuZ to combine learning with practical experience in APIs, DOM manipulation, and CSS animations, while building something useful.
